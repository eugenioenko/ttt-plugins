# lsp-markdown

Auto-configures `marksman` as the Markdown language server for ttt.

## Requirements

Install the language server binary:

```sh
See https://github.com/artempyanykh/marksman for installation instructions
```

## What it does

When installed, this plugin sets `lsp.servers.markdown` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Markdown files.

When uninstalled, the setting is automatically removed.
