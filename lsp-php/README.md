# lsp-php

Auto-configures `phpactor` as the PHP language server for ttt.

## Requirements

Install the language server binary:

```sh
See https://phpactor.readthedocs.io/ for installation instructions
```

## What it does

When installed, this plugin sets `lsp.servers.php` in your settings to enable LSP features (autocomplete, hover, diagnostics) for PHP files.

When uninstalled, the setting is automatically removed.
