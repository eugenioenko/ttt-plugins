# lsp-lua

Auto-configures `lua-language-server` as the Lua language server for ttt.

## Requirements

Install the language server binary:

See https://github.com/LuaLS/lua-language-server for installation instructions

## What it does

When installed, this plugin sets `lsp.servers.lua` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Lua files.

When uninstalled, the setting is automatically removed.
