# lsp-ruby

Auto-configures `ruby-lsp` as the Ruby language server for ttt.

## Requirements

Install the language server binary:

```sh
gem install ruby-lsp
```

## What it does

When installed, this plugin sets `lsp.servers.ruby` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Ruby files.

When uninstalled, the setting is automatically removed.
