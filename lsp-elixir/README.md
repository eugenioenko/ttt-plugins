# lsp-elixir

Auto-configures `elixir-ls` as the Elixir language server for ttt.

## Requirements

Install the language server binary:

```sh
See https://github.com/elixir-lsp/elixir-ls for installation instructions
```

## What it does

When installed, this plugin sets `lsp.servers.elixir` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Elixir files.

When uninstalled, the setting is automatically removed.
