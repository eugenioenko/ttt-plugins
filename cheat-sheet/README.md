# Cheat Sheet

Fetch programming cheat sheets from cheat.sh and display them in an editor tab.

## Commands

| Command | Keybinding | Description |
|---|---|---|
| `Cheat Sheet: Search` | `Ctrl+K H` | Focus the search input in the sidebar |

## Usage

Open the Cheat Sheet sidebar tab, type a query (e.g. `go/slice`, `python/list`, `bash/find`) and press Enter. Results are displayed in an editor tab and cached for quick re-access.
