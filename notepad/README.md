# Notepad

Persistent scratchpad for jotting down notes per workspace.

## Commands

| Command | Description |
|---|---|
| `Notepad: Clear All Notes` | Delete all notes (with confirmation) |

## Usage

Notes are stored in `.ttt-notepad.json` in the workspace root. Type a note in the input field at the bottom panel and press Enter. Right-click a note to delete it.
