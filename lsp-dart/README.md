# lsp-dart

Auto-configures `dart language-server` as the Dart language server for ttt.

## Requirements

Install the language server binary:

```sh
See https://dart.dev/get-dart for installation instructions
```

## What it does

When installed, this plugin sets `lsp.servers.dart` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Dart files.

When uninstalled, the setting is automatically removed.
