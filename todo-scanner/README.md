# TODO Scanner

Scans workspace for TODO, FIXME, HACK, and NOTE comments.

## Commands

| Command | Description |
|---|---|
| `TODOs: Scan Workspace` | Rescan the workspace for comment markers |

## Bottom Panel Shortcuts

| Key | Action |
|---|---|
| `R` | Refresh scan |

## Requirements

Requires [ripgrep](https://github.com/BurntSushi/ripgrep) (`rg`) to be installed.
