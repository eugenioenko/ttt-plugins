# JSON Viewer

Preview JSON files as an expandable tree.

## Commands

| Command | Description |
|---|---|
| `Preview: JSON` | Parse the current buffer as JSON and open a tree view in a new tab |

## Usage

Open a `.json` file, then run `Preview: JSON` from the command palette (`Ctrl+P`). The JSON structure is displayed as a collapsible tree with type badges (`str`, `num`, `bool`, `{N}`, `[N]`).
