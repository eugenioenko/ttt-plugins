# lsp-zig

Auto-configures `zls` as the Zig language server for ttt.

## Requirements

Install the language server binary:

See https://github.com/zigtools/zls for installation instructions

## What it does

When installed, this plugin sets `lsp.servers.zig` in your settings to enable LSP features (autocomplete, hover, diagnostics) for Zig files.

When uninstalled, the setting is automatically removed.
