# Go Test Runner

Run Go tests and view results in a sidebar panel.

## Commands

| Command | Keybinding | Description |
|---|---|---|
| `Go Tests: Run All` | `Ctrl+K T` | Run all tests in the workspace |
| `Go Tests: Run Current File` | | Run tests for the current file's package |
| `Go Tests: Refresh` | | Re-scan for test packages |

## Sidebar Shortcuts

| Key | Action |
|---|---|
| `R` | Refresh test list |
| `F` | Run selected test or package |
| `A` | Run all tests in the selected package |
| Right-click | Context menu |
