# Docker Manager

Manage Docker containers, images and volumes from the sidebar.

## Commands

| Command | Keybinding | Description |
|---|---|---|
| `Docker: Refresh` | `Ctrl+K R` | Refresh all Docker resources |
| `Docker: Prune Containers` | | Remove all stopped containers |
| `Docker: Prune Images` | | Remove all unused images |
| `Docker: Prune Volumes` | | Remove all unused volumes |

## Sidebar Shortcuts

| Key | Action |
|---|---|
| `R` | Refresh all |
| `Enter` | Expand / collapse |
| Right-click | Context menu (start, stop, restart, remove) |

## Requirements

Docker must be installed and the `docker` CLI must be available in your PATH.
